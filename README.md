# 📊 Sales Dashboard Project

## 🔹 Description
This project is a sales dashboard built using Python.

## 🔹 Tools Used
- Python
- Pandas
- Matplotlib / Plotly
- Excel Dataset

## 🔹 Features
- Sales analysis
- Data visualization
- Interactive dashboard in browser

## 🔹 How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the project:
   python dashboard.py
